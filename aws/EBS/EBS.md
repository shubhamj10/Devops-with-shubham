## EBS (Elastic Block Storage)

* Block based storage
* Runs ec2 OS, store data from db, file data, etc.
* Place in specific AZ. Automatically replicated withing the AZ to protect from failure.
* Snapshot is backup of a volume.

---
## EBS Types
* **General Purpose (SSD)** 
    - **most Work Loads**

* **Provision IOPS**
    - **Large Database** 

* **Throughput Optimized HD**
    - **Big Data and Data Warehouses**
